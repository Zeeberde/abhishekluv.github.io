Orchard CMS Mumbai Workshop : Beginner to Advanced

How will you benefit from this workshop?

You will get a good working knowledge of Orchard CMS and its admin panel and how to use Orchard CMS as an advance user.

After completing this workshop you will be able to download Orchard, Install it locally and remotely on a live web server, use the Orchard admin panel very efficiently, Install new themes and modules from the gallery, create pages, blogs and pretty much everything that a content manager will do to manage his content using Orchard CMS. After understanding the basics of Orchard we’ll dive into more advance topics and features like Content Types, Content Parts, Content Fields, Content Items, Taxonomies, Queries and Projector, Zones, Widgets, Layers, Layer Rules, Navigation, Recipes, Workflows and many more..

Orchard Beginner : Custom Forms in Orchard (External Post)

Welcome to Orchard Beginner! Custom Forms feature in Orchard allows us to create custom forms like contact forms or content contributions.

The Orchard.CustomForms Module/Feature is disabled by default and has dependencies on other modules like Contents, Orchard.Tokens, Orchard.Workflows.
