Welcome to Orchard Beginner

Jul 26th, 2014

Orchard Beginner is created with the aim to provide users and developers to have a Beginner’s Guide to learn the ins and outs of Orchard.

We begin with the Basics of Orchard i.e. Talking about Basic site management, User management and then we step into more important features of Orchard like Queries, Projector, Taxonomies, Custom Forms, Workflows and Recipes.

After that we’ll talk about how to Get Started with Theme Development in Orchard and later on Module Development too.

Again Welcome To Orchard Beginner.
