Learn How to Host Static Websites Using GitHub Pages(Udemy Course)

GitHub Pages are public webpages freely hosted and easily published through the GitHub website. You can create and publish them online using the Automatic Page Generator. If you prefer to work locally, you can use the GitHub for Mac and Windows apps.

In this course you’ll learn how to host a static website using GitHub Pages at no cost. You’ll need to purchase your own Domain Name which costs around $10 to $12 yearly that’s it.

A lot of people who are scared to get their First Website online now can have their website online right away using GitHub Pages.

In this course you’ll learn the following things:

  • Getting Started with GitHub
  • How to create Repositories and Organizations
  • How to use GitHub Pages
  • Types of GitHub Pages
  • How to use the Git Extensions Software
  • How to Host Static Websites using GitHub Pages

Note : Course is about GitHub Pages only and not about Designing Websites. This course only covers the Hosting part and not Designing.
